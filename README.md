#### 🏥 [FPGA-Based Smart Health Monitoring System](https://github.com/Ali0904/FPGA-Based-Smart-Health-Monitoring-System-Using-RISC-V-Core)
> **Best FYP Award — 3rd Career Fest (Open House 2026)**

Built a complete RISC-V core on Digilent Basys3 Artix-7 FPGA @ 100 MHz (145 mW).  
Integrated temperature, SpO2, and heart rate sensors via SPI/I2C into a memory-mapped peripheral system.  
Developed embedded C firmware for real-time acquisition, digital filtering, and safety alerts.
